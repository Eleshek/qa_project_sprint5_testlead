# QA Project: Sprint 5 - Test Lead Portfolio

Учебный командный проект в рамках курса по QA.  
Команда тестировала три веб-приложения: **Foodgram / Продуктовый помощник**, **Kittygram** и **Stellar Burgers**.

В проекте я выполняла роль **QA Engineer / Test Lead**: организовывала процесс тестирования, готовила шаблоны тестовой документации, распределяла задачи, контролировала сроки, качество тестовых артефактов и общий статус проекта.

## Моя роль

**QA Engineer / Test Lead**

Команда состояла из 10 человек.  
Проектный объем по трем веб-приложениям был выполнен за 1 неделю при изначальном сроке 1 месяц.

## Что входило в мои задачи

- Организация процесса тестирования по трем веб-приложениям.
- Подготовка шаблонов тестовой документации.
- Подготовка диаграмм, карт и вспомогательных материалов для анализа продукта.
- Распределение задач между участниками команды.
- Контроль сроков и статусов выполнения задач.
- Анализ требований и выявление неясностей.
- Контроль покрытия требований проверками.
- Проверка качества чек-листов, тест-кейсов и баг-репортов.
- Координация функционального, регрессионного и UI-тестирования.
- Контроль результатов проверок в разных браузерах.
- Сбор общего статуса по результатам тестирования.
